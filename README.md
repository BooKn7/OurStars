# OurStars - プレイロジックソースコード

このリポジトリは、現在制作中のゲーム「OurStars」の主要なプレイロジックやギミックを制御するC#スクリプトをまとめたものです。

## 概要

「OurStars」は、2人のプレイヤー（P1/P2）がお互いの引力や拘束機能を駆使して進む、協力型のアクションパズルゲームです。
このリポジトリには、プレイヤーの基本移動、引力による拘束アクション（RTボタンでの円軌道生成）、アイテムの取得や各種ギミックの挙動制御など、ゲームのコアとなるロジックの実装が含まれています。

## 主な構成

*   **`Player/`**: プレイヤーの基本操作や、相互に影響を与え合う引力アクションの制御システム。
    *   `PlayerMove.cs` / `Player2Move.cs`: P1とP2の基本的な移動とジャンプ処理。
    *   `CircularPathP1.cs` / `CircularPathP2.cs`: 引力アクションの設計コア。P1またはP2がRT入力で相手を一定の軌道上に拘束し、円運動させる制御を行います。エネルギーの消費計算や、失敗時の演出も管理しています。
    *   `PlayerTrapHandler.cs`: トラップ接触時のリスポーン処理と無敵時間の管理。
*   **`Items/`**: ステージ内に配置される各種アイテムの制御。
    *   `Coin.cs` / `CoinManager.cs`: コインの取得と、全回収時のギミック発動（オブジェクトの置き換え等）。
    *   `EnergyPickup.cs`: 引力アクションに必要なエネルギーを回復させるアイテム処理。
*   **`Trap/`**: トラップやギミックの挙動制御。
    *   `SpikeheadSpawner.cs` / `SpikeheadMovement.cs`: 一定間隔で飛んでくるトゲの生成と移動処理。
    *   `MoveToPoints.cs`: 指定したポイント間を自動で巡回移動するギミックの管理。
*   **イベント・システム関連**
    *   `GameManager.cs`: プレイヤーのリスポーン位置などのゲームシステム管理。
    *   `LoadSceneTrigger.cs`: 2人が同時にゴールに到達した際のシーン遷移判定。
    *   `MusicManager.cs`: シーンごとのBGMの切り替え（シーン間で独立した音楽管理）。
